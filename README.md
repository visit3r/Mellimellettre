# Mellimellettre

Ce SHELL a été créer presque dans sa totalité avec GNU BASH 5.1

et GNU nano version 5.4.

Ce bourne shell est un jeu qui est inspiré par des grilles de jeu 

de mot. Ce bourne shell est un jeux intéractif où tu doit tapper

dans l'ordre les lettres pour chaques mots avec le clavier. C'est 

un jeu de lettres mêlées.

Les variables $1,$2,$3,$... ne sont pas utilisées.

Ce jeux créer un log dans le même répertoire que le shell et qui

 écrit le surnon du joueur, l'ages du joueur, l'heure de début 

de session et l'heure de fin de session de jeux.
